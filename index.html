Git Cheat Sheet

cd folder gør så terminalen ved hvilken mappe/repositori der arbejdes i

git clone url downloader/kloner online filer ned på ens lokale computer

git status hvor er vi i processen? Er der nogle filer, som er blevet slettet eller tilføjet?

git add .her tilføjer vi de nye filer, vi gerne vil have registreret

Git commit -m “message”Her kan man skrive en besked til de andre, så de ved hvad man har opdateret i filerne. Git går ind og tager et billede af det og gemmer det i sin database.

Git pushSidste ting man skal gøre. Gør at filerne kommer online.

Git pullHvis men henter noget online ned til at blive lokalt(her har du allerede filerne lokalt,men de skal blot opdateres)
